Lab 03

Content Learned

CSS (Cascading StyleSheet) is a language that uses selectors, attributes and values to tell a document what it should look like. I have had use of it before but with this class I gained a deeper understanding of what it was aside from being a language that just makes things look good. Especially learning more about selectors and how they can be used, such as ID selectors, Class selectors and just the 'Simple' Element selectors.

Knowing more about what makes up CSS has given me more tools to apply to creative practice so that I can further master this language and create websites that are pleasing to look at.
